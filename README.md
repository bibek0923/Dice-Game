
# Rolling Dice App
This repository contains code for simple dice app using dart/flutter. 



## Features


- Dice Rolling Game
- Dice Rolling Screen
- Animated Dice
- Guess Feedback
- Play again option
- Fullscreen mode
- Cross platform
- Screen navigation





## To Run The App

 Run following command in terminal :

-  git clone https://github.com/bibek0923/Dice-Game.git 
- flutter run
